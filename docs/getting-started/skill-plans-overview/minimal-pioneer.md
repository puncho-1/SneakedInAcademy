# Pioneer - Minimal Skill Plan

Ziel: so schnell wie möglich sinnvoll in eine Pioneer steigen und Ore minen.

## Pflicht

- Mining III
- ORE Hauler I
- Mining Frigate III
- Mining Destroyer I

## Sehr sinnvoll direkt danach

- Mining Destroyer II
- Astrogeology III
- Survey III
- Shield Operation III
- Shield Management III
- Hull Upgrades III
- CPU Management IV
- Power Grid Management IV
- Navigation III

## Module / Praxis

Für ein simples Einsteiger-Setup sinnvoll:

- Mining Laser Upgrade I
- Survey Scanner I
- kleine Shield-Tank-Module
- Mining Laser / passende Anfänger-Module

## Priorität

1. Mining III  
2. Mining Frigate III  
3. ORE Hauler I  
4. Mining Destroyer I  
5. Mining Destroyer II  
6. Astrogeology III  
7. CPU Management IV / Power Grid Management IV  
8. einfache Shield-Skills auf III
